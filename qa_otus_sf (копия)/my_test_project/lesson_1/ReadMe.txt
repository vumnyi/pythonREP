Инструкция по запуску

Клонируем проект с гитхаба
Создаём виртуальное окружение
Устанавливаем зависимости из файла requirements.txt
Открываем проект в PyCharm
Устаналиваем Pytest в качестве runner
Запускаем тесты командой pytest test_introduction_to_pytest.py -s из виртуального окружения, находясь в папке lesson_1 с тестами
