Для создания и запуска тестов использовалось:

Python 3.6
PyCharm Community Edition 2017.1.5
Google Chrome 72.0.3626.81 (chromedriver 2.46)

Необходимые библиотеки:
Selenium WebDriver
Фреймворк Pytest


В файле selenium_fixture находится фикстура для запуска, открытия и закрытия окна браузера
В файле test_shop находится непосредственно тест, который можно модифицировать по желанию и передавать разные параметры для поиска значений, а так же дополнить другими тест-кейсами, без необходимости разбираться в том, как работаю тестовые методы, что позволяет их писать людям без знания языка программирования, так же в этом файле проводится проверка на идентичность названия модели в начале списка и в конце, при разных условиях сортировки
В файле application находятся тестовые методы, и логика описывающая их работу, а так же ожиданя появления элементов
